LIMITER


Limiter is a real time project that displays the parking lot of the institute campus retrieving data with the help of online server. The application helps the user get a hassle free parking experience. It shows the present parking state to the user, notes and notifies the driver through email once their allotted time is done. User can register their vehicles for future use. The admin can add new slots, edit FAQ and also set the maximum time for a vehicle type.
